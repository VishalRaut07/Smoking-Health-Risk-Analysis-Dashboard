-- Smoking Health Risk Analysis – Annual Dashboard
-- Project Overview

This project analyzes the annual health risks associated with smoking behavior across different demographics and age groups. The goal is to identify patterns in smoking status, duration, daily intake, and their impact on health indicators such as cholesterol and hypertension risk.

The dashboard provides interactive insights to help understand how smoking habits influence long-term health outcomes.

 --  Objectives

Analyze smoking status distribution (Never, Current, Former)

Compare smoking patterns across gender

Study smoking duration & daily intake trends by age group

Evaluate cholesterol and hypertension risk across age groups

Provide actionable health risk insights

 -- Tools & Technologies Used

Power BI – Dashboard creation & visualization

SQL – Data extraction & aggregation

Python (Pandas, NumPy, Matplotlib) – Data cleaning & preprocessing

Excel – Initial data exploration
